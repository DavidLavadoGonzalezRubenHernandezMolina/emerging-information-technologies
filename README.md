## 📂 Project

### 1️⃣ Steganography & Steganalysis Tool

🔍 **Image-Based Steganography and Steganalysis Web Application**

This project implements a full-stack web application focused on cybersecurity, specifically on the technique of steganography applied to digital images.

The tool allows users to hide secret messages inside images using the **LSB (Least Significant Bit)** technique, as well as detect and extract hidden information through forensic analysis.

---

### ⚙️ Features

* Hide text messages inside images using LSB steganography
* Extract and detect hidden messages from suspicious images
* Forensic analysis: EXIF metadata, color channel histograms and statistical anomaly detection
* Intuitive web interface built with React
* REST API backend built with Python and FastAPI

---

### 🛠 Technologies Used

* **Frontend:** React, JavaScript
* **Backend:** Python, FastAPI
* **Libraries:** Pillow, NumPy, piexif
* **Tools:** Git, VS Code

---

### 🚀 How to Run

**Backend:**
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

**Frontend:**
```bash
cd frontend
npm install
npm start
```
